Exam Results Declaration

A web-based application for managing and announcing examination results efficiently, designed with a secure admin dashboard and a student portal.

The Results Declaration System is an efficient platform for educational institutions to release examination results online. Students can easily access their results via institution-provided links formatted for specific exams, such as "I B.Tech I Sem (R23) Supplementary Examination." The system also includes an admin panel where administrators can generate result links, manage student data, and upload results via CSV files. Additionally, the system features a student performance block for viewing overall performance based on roll number.

Features:

  • Student Portal:
    • Access examination results via specific institution-generated links.
    • View results for different academic years and courses.
  • Admin Panel:
    • Generate result links based on examination type and format.
    • Manage student data, including name, date of birth, and phone number.
    • Upload student results using CSV files.
    • View overall student performance by entering the roll number.

Eduhub

Eduhub is an online learning platform designed to offer a comprehensive learning experience through features like course enrollment, progress tracking, and interactive quizzes. Developed as a collaborative project during an internship at Infosys, I contributed 85% of the frontend development.

Eduhub provides a flexible and interactive learning environment with a user-friendly interface. Key features include:

  • Course Catalog: Explore and enroll in various courses across different industries.
  • Progress Tracking: Keep track of your progress and completion status for enrolled courses.
  • User Profiles: Create and manage your user profile with personalized settings.
  • Authentication: Secure sign-up, login, verification, and session management.
  • Category Management: Courses are organized into categories for easy navigation.
  • Course Management: Includes structured modules and content with progress tracking.
  • Enrollment System: Allows students to enroll in courses and track their progress.

Website Structure:

  • Home Page: Menu options for easy navigation, About section, footer with social media links, contact information, and more.
  • Category Page: Displays course categories for users to search and select from.
  • Enrollment Page: Shows available modules for each course; enrollment restricted to logged-in users.
  • Dashboard Page: Displays all courses a user has enrolled in and completed.
  • Register & Login Pages: Handles user registration and login, ensuring secure access.
  • Reset Password Page: Allows users to update their password if forgotten.
